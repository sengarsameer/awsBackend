# awsBackend

## Project setup
```
npm install
```
* Create a "key.js" file in the root directory of this project, and write the below code in that file
  ```javascript
    module.exports = {
        iam_access_id: '<YOUR AWS ACCESS ID>',
        iam_secret: '<YOUR AWS SECRET KEY>'
    };
  ```

### Run on development
```
npm run dev
```

### Run for production
```
npm run start
```
